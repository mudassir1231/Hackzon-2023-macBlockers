# quantumx 2022 MacBlockers - Diabetic Retinopathy 
## Introduction
**Diabetic Retinopathy** is a condition which develops in the eye, which if not treated at early stages, could lead to permanent blindness. India is said to be the diabetic capital of the world by 2030 with over **80 million** people affected by it. If the blidness causing factor is identified early, this is a blindness which can be avoided.

### Methodology Used
In this project we have trained our own model using the dataset downloded from ISBI IEEE opensource which consists of 2256 images collected from retinal specialist Eye Clinic located in Department of Opthamalogy, Shanghai, China. Our front end user interface is a simple website where the patient can register and upload the eye scans.
The uploaded eye scan is analysed and appropriate output is rendered an email notification about the patients report details is sent to the registered mail.
